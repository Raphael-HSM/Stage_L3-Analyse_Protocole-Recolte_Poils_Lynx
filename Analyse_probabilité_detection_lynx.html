# Fonction de calcul des indices de surdispersion

overdisp_fun <- function(model) {
  rdf <- df.residual(model)
  rp <- residuals(model,type="pearson")
  Pearson.chisq <- sum(rp^2)
  prat <- Pearson.chisq/rdf
  pval <- pchisq(
    Pearson.chisq, 
    df=rdf, 
    lower.tail=FALSE
  )
  c(
    chisq=Pearson.chisq,
    ratio=prat,
    rdf=rdf,
    p=pval
  )
}

Calcul d’un indice de dispersion

On calcule un indice de dispersion pour vérifier s’il y a une surdispersion des données et adapter le modèle le cas échéant.

overdisp_fun(modele_ombrelle)
##       chisq       ratio         rdf           p 
## 726.7906329   0.9353805 777.0000000   0.9006369

On obtient une p-value>0.05, donc il n’y a pas de sur-dispersion de notre modèle ombrelle. De plus, on faisant ce test sur tous les modèles sélectionnés, on observe qu’il n’y a pas non plus de sur-dispersion des données. Le test est également négatif pour modele_anova.

Prise en compte des interactions

Au final, il y a eu 3 paramètres qui on eu des effets significatifs sur la détection de poils de lynx : période (PE), ecart_attractif et typelieu. Il va donc falloir tester les intercations : - PE:ecart_attractif - PE:typelieu - ecart_attractif:typelieu

On rajoute donc à notre modèle ombrelle les interactions entre les paramètres ayant été montrés significatifs par l’anova :

modele_final <- glmmTMB(

    typelieu:PE +
    (1 | MS),
