# Fonction de calcul des indices de surdispersion

overdisp_fun <- function(model) {
  rdf <- df.residual(model)
  rp <- residuals(model,type="pearson")
  Pearson.chisq <- sum(rp^2)
  prat <- Pearson.chisq/rdf
  pval <- pchisq(
    Pearson.chisq, 
    df=rdf, 
    lower.tail=FALSE
  )
  c(
    chisq=Pearson.chisq,
    ratio=prat,
    rdf=rdf,
    p=pval
  )
}

Création du modèle ombrelle

La variable réponse étudiée est la détection ou non de poils de lynx lors d’un relevé. C’est une variable binaire (0,1) qui suit une loi de Bernouilli. On doit donc utilisé un modèle linéaire généralisé (GLM). On prend pour effets fixes :

  • le type de piège à poil (barbelé, brosse)

  • le type de support pour le piège à poil (arbre, piquet, rocher)

  • le type d’odorant utiliser (sans odorant, Castoreum, lavande)

  • la période de l’année (avant, pendant et après le rut)

  • le type de micro-site (borne de marquage, lieu de passage, lieu sélectionné aléatoirement)

  • l’écart entre la date de relevé et la date de pose de l’odorant (valeur numérique)

On a écartant du facteur “Odorant” les données liées à l’odorant “Urine” car aucune détection de Lynx n’a été observé, ce qui menait à des problèmes de convergence dans le modèle. On doit tenir compte de l’effet aléatoire de l’identité du micro-site, c’est pourquoi on emploie un modèle linéaire mixte généralisée (GLMM) avec l’identité de micro-site en effet aléatoire.

modele_ombrelle <- glmmTMB(
  DetectionLynx ~ 
    TypePiege +
    Support +
    Odorant +
    TypeLieu +
    PE +
    (1 | MS),
  data = Tidy_Releves_PP,

On observe qu’il y a une différence significative entre la P1 et les 2 périodes P2 et P3, mais que celles-ci n’ont pas de différences significatives entre elles. De même, on observe pas de contraste significatif entre les différents type de support. Enfin, il y a une différence significative entre les pièges à poil en barbelé et en brosse.

On en déduit donc que les pièges à poil en barbelé sont les plus efficaces, et qu’il est préférable d’effectuer la récolte de poil pendant et après le rut.
